# Casino Gambling Game

![https://github.com/SSoggyTacoMan/sprig/blob/main/games/img/GamblingCasino.png?raw=true](https://github.com/SSoggyTacoMan/sprig/blob/main/games/img/GamblingCasino.png?raw=true)

### A 6-in-1 casino arcade built for the Hack Club Sprig engine. It features 6 games sharing a balanced good unified economy, global jackpot, and dynamic betting tiers.

#### The Games

* Slots: 3-reel machine with an optional 50/50 double-or-nothing risk roll.

* High/Low Cards: Guess the next card. Ties pay out big.

* Spin Wheel: Switch between Safe (frequent small wins) and Hot (high risk/reward) modes.

* Blackjack: Standard 21 against the dealer.

* Roulette: Interactive betting board for numbers, colors, and odds/evens.

* Bingo: Get a boll, mark it on your csrd, get 5 in a row. With cool stuff like.  (currently overpowered, so ig WIP, but the game works)


#### Mechanics

* Global Jackpot: A percentage of every bet and win feeds a universal jackpot.

* Auto-Scaling Bets: Betting options automatically adjust based on your current bankroll (from 5 to 50,000+).

* Fast Pacing: We rigged the odds slightly in your favor. If you're on a cold streak in Bingo, the game steps in and guarantees a 		match to keep things exciting.

#### Controls

* WASD: Move cursor or adjust bet size.

* J / I / L: Main action keys (Spin, Hit/Stand, Select).

* K: Back / Menu.
